# Proposal Template

- Title: Short descriptive title
- Authors: GitHub username(s) / wallet address(es)
- Date: YYYY-MM-DD

## Summary
One-paragraph summary of the proposal.

## Motivation
Why this proposal is necessary and what problem it solves.

## Specification
Detailed description of the change, including smart contract code if applicable, on-chain parameters, and off-chain actions.

## Implementation
Steps required to implement the proposal, including timeline and responsible parties.

## Security considerations
Any risks and proposed mitigations.

## Voting options
- For — implement the proposal as written
- Against — do not implement
- Abstain — no opinion

## References
Links to related discussions, design docs, or external resources.
